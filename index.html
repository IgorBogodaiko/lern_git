до папки git_test я підключу git, для цього меті потрібно в даній папці відкрити консоль гітбаш та ввести команду git init

git init - ініціалізуєм гіт, грубо кажучи, поселяємо його в папку з проектом. щоб подивитися які файли попали в гіт потрібно ввести команду git status

git add . добавить всі файли в область StageArea з якої гіт почне слідкувати за змінами та файлами

git commit -m 'кометн який ми лишаємо' добавляє файли для постійного збереження, а також коментар

git log подивития коміти та коменти

git commit -a -m 'comment text' команда яка за один раз добавляє зміни що сталиса в файлі після редагування а також робить комміт та комент до нього, по суті це скорочений запис команд адд та комміт
